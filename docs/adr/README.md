# ADRs (Architecture Decision Records)

Use this folder to store accepted architecture decisions.

## Naming

Use incremental IDs:
- `0001-short-title.md`
- `0002-short-title.md`

## Process

1. Copy from `docs/adr-template.md`
2. Propose in PR with context and alternatives
3. Mark status (`Proposed` or `Accepted`)
4. Link decision from relevant docs
