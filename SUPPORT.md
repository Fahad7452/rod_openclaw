# Support

## Getting Help

Before opening a new issue:
- Read `README.md`
- Check `docs/getting-started.md`
- Review `docs/faq.md` if available in your active docs set

## Where to Ask

- Use issue templates for bugs and feature requests
- For security concerns, follow `SECURITY.md`

## What to Include

- What you tried
- Expected behavior
- Actual behavior
- Environment details
- Relevant logs or screenshots (if safe to share)
